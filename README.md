# The Marauder's Chart

An interactive data visualisation of character mentions across all 7 Harry Potter books.

Every page. Every line. Each mark is an approximate mention — positioned by page (left → right) and line (top → bottom).

**[Try it live on GitHub Pages](https://nebinbiju1.github.io/the-marauders-chart/)** — or download `index.html` and open it directly in any browser, no server needed.

## How it works

- **X-axis** — page position across all 3,407 pages of the series
- **Y-axis** — line on the page (1–40)
- **Dots** — each dot represents an approximate mention of that character
- Click a portrait to toggle a character on or off
- Hover over a dot to see the book, page, and chapter

## Data

Mention counts are fan-research estimates. The spatial distribution within each book is procedurally generated using character-specific density functions — for example, Voldemort barely appears in the early chapters of books 1–3, Snape spikes in the second half of *The Half-Blood Prince*, and Ron is nearly absent from the middle third of *The Deathly Hallows*.
